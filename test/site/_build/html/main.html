QLiteHtmlBrowser Test Site¶

This is the main test site for manual checks of the QLiteHtmlBrowser implementation. Describe what you WANT to see and check manually if its rendered correctly. This is helpful and required as long as there is no fully automated test engine for QLiteHtmlBrowser available.
